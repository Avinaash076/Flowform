## 🎯 JavaScript API

```javascript
// Animate single element
animateElement(element, 'cardSlideUp', 400);

// Stagger animate multiple elements
staggerAnimate('.list-row', 'listItemFadeIn', 50);

// Show loading spinner
showLoading();

// Hide loading spinner
hideLoading();

// Example: Animate on button click
button.addEventListener('click', () => {
    animateElement(button, 'buttonPulse', 500);
});
```

## 📱 Responsive Notes

- All animations work on mobile
- Respects `prefers-reduced-motion` setting
- 60fps on most devices
- GPU-accelerated for smooth performance

---

## ⚡ Performance Tips

1. **Use transform and opacity** - Avoid width/height changes
2. **Limit simultaneous animations** - Use stagger instead
3. **Test on low-end devices** - Ensure smooth performance
4. **Use utility classes** - Faster than custom CSS
5. **Combine effects** - Multiple classes for rich effects
